TL;DR

Mistral has open-sourced a new Mixture-of-Experts model that outperforms Llama 2 70B on most benchmarks while using significantly fewer active parameters.

Why it matters

  • check

    Cost reduction: Drastically reduces inference costs for local deployment making it viable for consumer hardware.

  • check

    Open challenge: Directly challenges the dominance of closed-source giants with comparable performance.

  • check

    New architecture: Introduces a sliding window attention mechanism that handles longer contexts more efficiently.
